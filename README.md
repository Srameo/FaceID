# FaceID
> a simple Face ID tool for normal camera (can't promise the security)


## Enviorment
- python 3.8.6
- opencv 4.4.0
- pyautogui 
- sleepwatcher 2.2.1 (for mac)
  > you can download this on this [bb's homeppage](https://www.bernhard-baehr.de).


## Usage
- just run the code below!
  - for setup
    ```
    ./setup.sh
    ```
  - for start
    ```
    ./start.sh
    ```
  - for clean
    ```
    ./setup.sh
    ```
